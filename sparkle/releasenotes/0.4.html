New in Frenzy 0.4

  • Major overhaul of the way replies work
  • Fixed bug where popup dialogs (such as the delete confirmation dialog) appeared behind the main Frenzy window
  • Fixed issue where feed updates were sometimes not detected
  • Removed redundant WebKit menu items such as 'Open link in new window' and 'Download linked file'
  • If no feed items are available, now shows 'No items' splash screen
  • Fixed bug where you could send a message with no shared folders active
  • Fixed issue where WebView scrollbar would flicker when showing the window
  • Fixed issue where Frenzy could run out of file descriptors and crash
  • Rewrote window activation/hide mechanism
