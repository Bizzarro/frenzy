New in Frenzy 0.5

  • Added file/folder sharing support
  • Added ability to change avatar and name in the preferences
  • Fixed bug where reply to item would be lost when backspacing
  • Added ApplicationConnectors to enable sharing from iTunes, iPhoto and Preview
  • Added support for sharing links from Google Chrome
  • Re-sharing an item already in the feed is now treated it as a reply to that item
  • Tweaked feed display visuals
  • Fixed bug where less than 25 items were being shown in the feed
  • Added more clock skew calculation time servers
  • Added hidden preference to only show buttons on mouse over
