What's New in Frenzy 1.0

  • Completely redesigned feed display UI
  • Support for Dropbox folders in sub-folders
  • Now supports drag & drop of files/folders/text/urls onto the menu item
  • Preliminary support for OS X Lion
  • Many improvements to the main UI, i.e the main window is now a non-activating panel and doesn't steal focus when sharing
  • Support for Firefox 5
  • URLs shared in messages are now clickable
  • Dozens of other bug fixes and tweaks
